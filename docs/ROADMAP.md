# Project Roadmap

This roadmap outlines planned research directions.  
Items are exploratory and subject to change.

---

## Near-Term (Research Expansion)
- Probabilistic state estimation (Kalman filtering)
- Improved physiological noise modeling
- Expanded simulation scenarios

## Mid-Term (System Integration)
- Hardware abstraction layer for infusion pumps
- Wearable sensor ingestion interfaces
- Real-time dashboard data streaming

## Long-Term (Validation & Translation)
- Machine-learning controller optimization
- Multi-patient comparative simulations
- Regulatory documentation preparation
